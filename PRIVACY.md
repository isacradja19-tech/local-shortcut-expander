# Easeit Privacy Policy

Easeit is a local text shortcut expansion extension.

## Summary

- No account is required.
- There is no backend server.
- There are no analytics.
- There are no ads.
- There is no tracking.
- No user data is sold.
- No user data is sent to third parties.

## Local Storage

Snippets are stored locally in your browser using IndexedDB.

Settings are stored locally in your browser. This includes small settings such as whether shortcut expansion is enabled.

## Export And Import

Exported JSON files are created only when you choose Export backup.

Imported JSON files are read only when you choose Import backup.

Easeit does not automatically upload, sync, or share exported or imported files.

## Typed Text

Easeit reads typed text only enough to detect local shortcut triggers, such as `/sig`.

When a trigger matches a saved local shortcut, Easeit replaces the trigger with the saved snippet text.

Password fields are ignored.

## Third Parties

Easeit does not send snippet content, typed text, settings, or backup files to any third party.
